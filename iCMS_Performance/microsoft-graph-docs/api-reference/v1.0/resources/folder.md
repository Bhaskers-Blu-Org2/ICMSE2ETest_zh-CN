# <a name="folder-resource-type"></a>文件夹的资源类型

该**文件夹**资源分组到一个单一的结构与文件夹相关的数据，在一个项目上。



## <a name="properties"></a>属性

| 属性       | 类型  | 说明                                                     |
|:---------------|:------|:----------------------------------------------------------------|
| **childCount** | Int64 | 直接包含在此容器的子级的数目。 |
