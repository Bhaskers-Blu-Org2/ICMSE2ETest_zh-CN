# <a name="get-locatedriskevent"></a>获得 locatedRiskEvent

检索的属性和关系的 locatedriskevent 对象。
## <a name="prerequisites"></a>先决条件
执行此 API 所需的以下**范围**︰ *IdentityRiskEvent.Read.All*
## <a name="http-request"></a>HTTP 请求
<!-- { "blockType": "ignored" } -->
```http
GET /locatedRiskEvents/<id>
```

## <a name="request-headers"></a>请求标头
| 名称      |说明|
|:----------|:----------|
| 授权  | 持有者<code>|
| 工作簿会话 Id  | 工作簿会话 Id 确定更改都不具有持久性。 可选项。|

## <a name="request-body"></a>请求正文
请不要提供此方法请求正文。
## <a name="response"></a>响应
如果成功，此方法返回`200 OK`响应代码和[locatedRiskEvent](../resources/locatedriskevent.md)对象在响应正文中的。
