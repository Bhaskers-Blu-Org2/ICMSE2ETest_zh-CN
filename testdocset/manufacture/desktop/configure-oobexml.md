# <a name="configure-oobexml"></a>配置 Oobe.xml


Oobe.xml 是用来收集文本和图像的自定义 Windows® OOBE 的内容文件。 若要生成一个 Windows 映像包含多语言交付到多个国家或地区，您可以添加多个 Oobe.xml 文件来自定义基于客户的语言和国家/地区选择的内容。
