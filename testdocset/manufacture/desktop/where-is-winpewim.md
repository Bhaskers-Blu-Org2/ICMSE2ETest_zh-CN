# <a name="where-is-winpewim"></a>WinPE.wim 在哪里


WinPE.wim︰ 在 Windows 7 中的主 Windows PE 启动文件已重命名从 winpe.wim 到 boot.wim。 此文件是在 Windows PE 中\\源文件夹。 可以与 WinPE.wim 相同的方式对其进行修改。
