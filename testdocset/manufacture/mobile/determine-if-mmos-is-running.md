# <a name="determine-if-mmos-is-running"></a>确定是否正在运行 MMO


您可以检查 MMO 是否运行检查在制造模式下运行时的方式相同。 有关详细信息，请参阅[检测制造模式](how-to-detect-manufacturing-mode.md)。

**请注意** 根据此项已过时，不应查询 ManufacturingOS 注册表设置。
