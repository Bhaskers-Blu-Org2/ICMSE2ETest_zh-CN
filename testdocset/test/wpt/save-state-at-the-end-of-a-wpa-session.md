# <a name="save-state-at-the-end-of-a-wpa-session"></a>保存在 WPA 会话的结束状态


您可以保存 Windows 性能分析器 (WPA) 会话的状态，以便以后您可以重新打开它。

**若要保存您当前的 WPA 会话的状态**

1.  在**文件**菜单上，单击**导出会话。**

2.  为文件指定一个名称，然后单击**保存**。

    WPA 文件文件夹中的文档文件夹中以.wpa 扩展名保存文件。
