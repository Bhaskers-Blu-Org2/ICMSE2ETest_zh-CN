# <a name="filename"></a>filename


此操作会生成报告总结了在跟踪文件的名称。

``` syntax
-a filename [-range T1 T2]
```

## <a name="options"></a>Options


<a href="" id="-ranget1-t2"></a>**-范围***T1 T2*  
显示记录之间*T1*和*T2*，以微秒为单位的文件名称。
