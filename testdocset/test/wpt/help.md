# <a name="help"></a>help


显示帮助 Xperf 选项和操作。

``` syntax
xperf -help [<option>|<action>]
```

## <a name="parameters"></a>参数


<a href="" id="-help--option---action--"></a>**-帮助***\[&lt;选项&gt;|&lt;操作&gt;\]*  
显示所指定的选项或操作的帮助信息。 如果未指定任何选项或操作，将显示可用的帮助主题的列表。
