# <a name="zoom-in-on-a-time-interval"></a>放大时间间隔


您可以放大反复要放大图表中的时间间隔。

**若要放大的时间间隔**

1.  将指针拖到图表中所需的时间间隔。

2.  用鼠标右键单击选定的时间间隔，然后选择**缩放到所选的时间范围**。

    可以多次重复第 2 步，若要进一步放大。

**若要返回到原始视图**

-   图中，用鼠标右键单击，然后选择**Unzoom**。
