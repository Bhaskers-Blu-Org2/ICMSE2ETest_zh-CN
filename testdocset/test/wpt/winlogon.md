# <a name="winlogon"></a>winlogon


此操作会生成一个文本文件，其中概述了为登录相关的度量标准。

``` syntax
-a winlogon [-range T1 T2]
```

## <a name="options"></a>Options


<a href="" id="-ranget1-t2"></a>**-范围***T1 T2*  
显示事件从*T1*到*T2*，以微秒为单位。
