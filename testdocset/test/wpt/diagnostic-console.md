# <a name="diagnostic-console"></a>诊断控制台


当您首次打开 Windows 性能分析器 (WPA)，但可以拖出到一个单独的窗口，也可以将它停放在顶部或侧面，诊断控制台窗口会显示在工作区的底部。 它列出有关分析工作流过程中发生的异常的信息。

您可以诊断符号解码问题通过此控制台。
