# <a name="pnp"></a>即插即用


此操作会产生一个文本文件，其中总结了插事件的衡量标准。

``` syntax
-a pnp [-range T1 T2]
```

## <a name="options"></a>Options


<a href="" id="-ranget1-t2"></a>**-范围***T1 T2*  
以微秒为单位显示*T1*和*T2*，之间的事件。
