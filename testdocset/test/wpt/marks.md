# <a name="marks"></a>标记


此操作会产生一个文本文件，其中总结了标记放在跟踪中。

``` syntax
-a marks [-summary] [-range T1 T2]
```

## <a name="options"></a>Options


<a href="" id="-summary"></a>**-摘要**  
显示跟踪中的标记。

<a href="" id="-ranget1-t2"></a>**-范围***T1 T2*  
从*T1*到*T2*，同时以微秒为单位的时间范围中显示仅标记。
