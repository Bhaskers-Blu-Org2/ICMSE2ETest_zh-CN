# <a name="analysis-assistant"></a>分析助理


**分析助手**选项卡是中心区的 Windows 性能分析器 (WPA)。 若要分析一个录制，可以从图形浏览器窗口拖到该选项卡拖动图形缩略图。 每个关系图的左侧是图例标签行或条形图上的图形以及关联的数据表。 模拟运算表以表格形式显示关系图的数据。
