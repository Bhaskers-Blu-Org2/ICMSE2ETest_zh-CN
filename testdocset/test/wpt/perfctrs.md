# <a name="perfctrs"></a>perfctrs


此操作会产生一个文本文件，其中总结了对性能计数器相关的度量标准。

``` syntax
-a perfctrs [-range T1 T2]
```

## <a name="options"></a>Options


<a href="" id="-ranget1-t2"></a>**-范围***T1 T2*  
显示性能计数器记录之间*T1*和*T2*，微秒。
