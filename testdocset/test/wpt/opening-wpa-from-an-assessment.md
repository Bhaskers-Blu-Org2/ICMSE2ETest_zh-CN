# <a name="opening-wpa-from-an-assessment"></a>从评估开始 WPA


从评估平台与评估结果页中，可以单击问题窗口打开 Windows 性能分析器 (WPA) 的评估结果中的**Windows 性能分析器**链接。 在 WPA 的问题窗口的右上角显示在评估平台中出现的相同问题。

在 WPA 的**分析**选项卡提供了在评估中发现的问题的摘要报告。
