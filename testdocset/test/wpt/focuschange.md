# <a name="focuschange"></a>focuschange


此操作会生成总结了 Z-顺序的窗口的焦点更改文本报告。

``` syntax
-a focuschange [-range T1 T2]
```

## <a name="options"></a>Options


<a href="" id="-ranget1-t2"></a>**-范围***T1 T2*  
显示窗口线程*T1*和*T2*，间中记录微秒的焦点更改事件。
