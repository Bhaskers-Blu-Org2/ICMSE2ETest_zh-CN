# <a name="mark"></a>标记


显示标记和标记刷新选项。

``` syntax
xperf -m | -mf <mark text>
```

## <a name="parameters"></a>参数


<a href="" id="-mstring"></a>**-m***字符串*  
在当前的跟踪日志标记。

<a href="" id="-mfstring"></a>**-mf***字符串*  
在当前的跟踪日志标记，而且，如果启用了占地面积，将刷新工作集。 此参数是仅供内部使用。
